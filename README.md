# acidroxygen

Shared roxygen2 documentation for [Acid Genomics][] packages.

## Installation

This is an [R][] package.

```r
install.packages(
    pkgs = "acidroxygen",
    repos = c("r.acidgenomics.com", getOption("repos"))
)
```

[acid genomics]: https://acidgenomics.com/
[r]: https://www.r-project.org/
